課程簡介

 

 

課程目標

 

◆瞭解 Java SE 平台的應用和相關的開發環境。

◆掌握 Java 程式語言的基本概念、語法結構和開發技術。

◆熟悉面向對象程式設計的基本概念和設計模式。

◆學習如何開發出簡單的應用程式。

◆增強解決問題和調試程式的能力。

 

上課前需要先準備

 

為了讓學員在課堂上能夠更好地理解和學習，建議學員在開始課程前先熟悉以下內容： Java SE 平台的基本概念和開發環境的使用方法； 基本的程式設計概念，例如流程控制、變數類型、陣列等； 熟悉面向對象程式設計的基本概念。
